# Nebula Run 🚀

A single-page HTML space shooter built from the uploaded PNG sprite pack.

## Play

Open `index.html` in a modern browser.

## Controls

- **Move:** `WASD` or arrow keys
- **Fire:** `Space`
- **Pause:** `P`, `Esc`, or the pause button
- **Touch:** on-screen joystick + shoot button

## Features

- Layered parallax nebula backgrounds from `PNG/Backgrounds`
- Animated player ship and enemy squadrons from `PNG/Main Planes` and `PNG/Enemy`
- Projectile, muzzle, turbine, and collision effects from the uploaded sprite folders
- Pilot sign-in on the briefing screen with name/country plus a small flag badge on the ship
- Touch-friendly HUD using the provided pause, analog, shoot, and health bar assets
- No build step or dependencies — everything runs offline from `index.html`
