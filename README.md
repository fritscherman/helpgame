# ExPlay 3 Quest — AV Configuration Challenge 🎮

An interactive HTML/JavaScript learning game that teaches **ExPlay 3** media-server configuration in a fun, gamified way.

## How to Play

Open `index.html` directly in any modern browser — no server or dependencies required.

## Game Overview

| # | Level | Topics covered |
|---|-------|----------------|
| 1 | 🖥️ Screen Setup | PROJECTION (2× FHD projectors) & MONITOR (WUXGA) |
| 2 | 📐 Field Configuration | Background + PiP fields, Cover vs Fit scaling |
| 3 | 🔄 Transformations | 45° PiP rotation, horizontal background flip |
| 4 | 🎬 Timeline & Loop | 3 videos + 3 stills, async cue, VIOSO test pattern, keyer |
| 5 | 📋 Playlists | Saturation correction, 2 videos + 2 stills × 8 s |
| 6 | ✨ Effects & Masks | Dome shader on PiP, V-shape mask, Client/Follow linking |
| 7 | 🎛️ User Panel & PJLink | Cue controls, saturation/audio sliders, PJLink ON/OFF |
| 8 | 📡 Data Track & TCP | 0–42 range, keyframes, TCP → 192.168.1.54:8567 |
| 9 | ⏰ Scheduler & Auto-Start | Tue 09:00 loop, Mon 17:00 cue 2, 04:00 daily restart |

## Features

- 🤖 **Exa** — animated robot guide character (inline SVG)
- ❤️ 3-lives system with instant feedback
- 💡 Hints available (small point deduction)
- 🗺️ Level map with star ratings and unlock progression
- 🎊 Confetti celebrations on correct answers & level completions
- 🏆 Final certificate screen with total score
- Multiple interaction types: multiple-choice, drag & drop, slider, text input
- Animated starfield background
- Zero external dependencies — runs offline from a single HTML file
